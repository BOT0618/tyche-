tyche User Remark Extension - Privacy Policy

Last Updated: December 2024

Important Notice: This extension does not collect, store, or transmit any personally identifiable information. All data is stored only in the user's local browser.

1. Data Collection

This extension only collects the following data:

  • Remark Data: Custom remarks added by users for blockchain addresses
  • Website Content: Address information from tyche.run pages (only for providing remark functionality)
  • Language Preference: User-selected interface language (Chinese/English)

2. Data Storage

All data is stored only in the user's local browser:

  • Uses browser's local storage functionality (chrome.storage.local)
  • Data is not uploaded to any servers
  • Data is not shared with third parties
  • Users can delete all data by removing the extension at any time
  • Data is saved only on the user's device and does not sync across devices

3. Data Usage

Collected data is used only for:

  • Providing remark functionality for users
  • Displaying user-added remarks on tyche.run pages
  • Managing user's remark list
  • Remembering user's interface language preference

4. Data Protection

We take the following measures to protect user data:

  • All data is stored locally and not uploaded to any servers
  • No personally identifiable information is collected
  • No user browsing behavior is tracked
  • No website access data is collected
  • No user browsing history is collected
  • No personal information is collected

5. Third-Party Services

This extension does not use any third-party services to collect or process user data. We only interact with the following services:

  • tyche.run: Only reads page content when users visit this website to provide remark functionality
  • Browser Storage API: Used for local storage of user remark data

6. User Rights

Users have the following rights:

  • Delete all data by removing the extension at any time
  • Clear extension storage data in browser settings
  • Complete control over their remark data
  • Export their remark data
  • Import remark data
  • Stop using the extension at any time

7. Policy Updates

If there are any updates to this privacy policy, we will notify users when the extension is updated. Users can learn about the latest policy through:

  • Visiting this privacy policy page
  • Checking the extension's update log

Data Retention: Since all data is stored only on the user's local device, we cannot access or delete user data. Users can completely delete all stored information by removing the extension or clearing browser data.
